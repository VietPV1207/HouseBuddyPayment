# HouseBuddy Payment App

## Overview
HouseBuddy is a PWA application with ReactJS frontend and NodeJS backend.

## Structure
- `client/` - ReactJS frontend (PWA)
- `server/` - NodeJS backend API

## Setup
1. Install client dependencies: `cd client && npm install`
2. Install server dependencies: `cd server && npm install`
3. Configure `.env` with MongoDB Atlas connection string
4. Run server: `npm start` (in server directory)
5. Run client: `npm start` (in client directory)
